## 数组-[自然数](../math/natural_number)
随意地在纸上写几个数，它们是无序的，那么要怎么使它们有序呢，我们便可以借助最自然的有序对象——自然数，只要把这些无序的数从0开始一一绑定，我们就获得了一个数组，比如：
$$a=[11,45,14,19,19,8,10]$$
绑定了 $3$ 的数就是 $19$，我们把数组中隐藏的自然数称为索引，这里的 $19$ 也就可以写作 $a[3]$

而很多时候这些数可能有一定的关联，我们会对它们进行分组，比如：
$$b=[[11,45,14],[19,19,8,10]]$$
这就成为了二维数组，原来的 $a[3]$ 到了这里就变成了 $b[1][0]$，如果不清楚具体的分组情况或分的组太多就统一称为多维数组

然而，在相当多的情况下，组与组之间也不是毫无关联的，它们的结构经常是相同的，这使得下面这种规则的二维数组更为常见：
$$c=[[11,45,14],[19,19,810]]$$
而将 $b$ 这种称为不规则二维数组，并且**以后默认"数组"的表述都是指规则数组，不规则数组需要明确指出**


::: tips
在物理和工程中从1开始要比从0开始常用，毕竟生活中没人会说"小明在比赛中获得了第0名"(除非这是什么计算机方面的比赛)，而且从1开始计数的结果n直接有总数的含义，这使得我们经常需要n-1以对上最后的索引
:::
